Bit_AutoTradingBot 이용약관

Version 1.0 | 시행일: 2025년

제1조 (목적)

본 이용약관은 Bit_AutoTradingBot(이하 "회사")이 제공하는 비트코인 자동매매 프로그램 서비스(이하 "서비스")를 이용함에 있어 회사와 이용자(이하 "회원") 간의 권리, 의무, 책임 사항 및 기타 필요한 사항을 규정함을 목적으로 합니다.

제2조 (서비스의 성격 및 작동 환경)

  • 본 서비스는 회원의 개인 PC에서 실행되는 자동매매 프로그램이며, 작동을 위해서는 회원의 PC가 인터넷에 상시 연결되어 있어야 합니다.
  • 서비스는 API 키를 이용해 회원이 지정한 거래소에서 주문을 실행하나, 거래소 API 오류, 서버 장애, 네트워크 불안정 등으로 인해 실행이 실패하거나 손실이 발생할 수 있습니다.
  • 회사는 회원의 개인 PC 환경 및 네트워크 상태를 통제할 수 없으며, 회원 PC의 설정, 보안, 인터넷 상태로 인해 발생하는 문제에 대해 책임을 지지 않습니다.

제3조 (비투자자문 고지)

중요 고지사항

  • 본 서비스는 투자자문, 투자일임, 자산운용, HTS, MTS 서비스가 아닙니다.
  • 이는 단순 자동화 도구이며, 매수·매도 결정에 대한 책임은 회원 본인에게 있습니다.
  • 회사는 투자 결과(수익률, 원금 보전, 전략의 지속성)를 보장하지 않습니다.
  • 과거의 수익률이나 백테스트 결과는 미래의 수익을 보장하지 않습니다.

제4조 (약관 동의)

  • 회원은 서비스 이용 전 본 약관에 동의해야 하며, 동의는 회사가 제공하는 온라인 동의 페이지를 통해 이루어집니다.
  • 모의투자에서 실전투자로 전환 시 별도의 동의 절차를 거쳐야 합니다.
  • 회원의 동의 기록(동의 시점, IP 주소, 약관 버전)은 서버에 보관됩니다.

제5조 (이용요금 및 구독 기간)

  • 서비스는 월 구독제로 운영됩니다. (1개월, 3개월, 6개월 단위)
  • 결제는 무통장 입금 방식으로 진행됩니다.
  • 구독 기간 만료 후 연장하지 않으면 프로그램은 자동으로 작동 중단됩니다.
  • 입금 확인 후 이용약관 동의 절차가 완료되어야 프로그램이 제공됩니다.

제6조 (환불 정책)

환불 안내

  • 결제일로부터 7일 이내 환불 요청 시 전액 환불
  • 결제일로부터 7일 경과 후에는 이미 제공된 서비스 기간에 대한 환불 불가
  • 환불 요청: bitautobot.help@gmail.com
  • 환불 처리 기간: 요청일로부터 영업일 기준 7일 이내

제7조 (이용자격)

  • 본 서비스는 만 19세 이상만 이용할 수 있습니다.
  • 회원은 서비스 신청 시 만 19세 이상임을 확인하는 절차에 동의해야 합니다.
  • 미성년자가 본인 또는 타인의 명의를 도용하여 서비스를 이용한 경우, 회사는 해당 계정을 즉시 해지할 수 있으며, 이로 인한 모든 책임은 해당 이용자에게 있습니다.

제8조 (면책 조항)

회사는 다음 사유로 발생한 손해에 대해 책임을 지지 않습니다

거래소 관련 문제
  • 거래소 서버 다운 또는 점검
  • API 오류, 지연, 거절
  • 체결 지연, 슬리피지
  • 거래소 규정 변경
회원 환경 문제
  • 회원 PC 전원 꺼짐 또는 재부팅
  • 인터넷 불안정 또는 끊김
  • PC 성능 부족 또는 프로그램 강제 종료
  • API 키 관리 소홀 또는 유출
전략/알고리즘 문제
  • 시장 변동성 확대에 따른 손실
  • 전략 미작동 또는 과거와 다른 성능
  • 프로그램 오류로 인한 비정상 주문
불가항력
  • 천재지변, 전쟁, 해킹, 비상경제상황, 법률/정책 변경

제9조 (회원의 의무)

  • 회원은 자신의 API 키를 스스로 발급·관리해야 하며, API 키 관리 소홀로 인한 피해에 대해 회사는 책임을 지지 않습니다.
  • 회원은 정상 작동을 위해 PC 및 인터넷을 안정적으로 유지해야 합니다.
  • 회원은 프로그램을 임의로 수정, 변조, 역공학, 배포할 수 없습니다.
  • 회원은 본인 명의의 계정만 사용해야 하며, 타인에게 계정을 양도하거나 공유할 수 없습니다.

제10조 (회사의 의무)

  • 회사는 프로그램의 정상적인 배포와 업데이트를 위해 노력합니다.
  • 회사는 서비스 제공 과정에서 취득한 회원의 정보를 관계 법령에 따라 보호합니다.
  • 회사는 서비스 장애 발생 시 신속한 복구를 위해 노력합니다.

제11조 (서비스 중단)

회사는 아래 상황에서 서비스 제공을 중지하거나 종료할 수 있습니다:

  • 시스템 점검 및 업데이트
  • 기술적 문제 발생
  • 법적 규제 또는 정부 정책 변화
  • 회사의 사업 중단

이 경우 회원에게 사전 통지하며, 긴급한 경우 사후 통지할 수 있습니다.

제12조 (손해배상 및 책임의 제한)

  • 회사는 어떠한 경우에도 회원의 투자 손실, 기회손실, 기대수익 미달 등에 대해 책임을 지지 않습니다.
  • 회사의 고의 또는 중대한 과실로 인한 손해가 인정되는 경우, 배상금액은 회원이 해당 서비스에 지불한 최근 1개월 구독료를 한도로 합니다.

기타 조항

  • 제13조 (계약의 해지): 회원은 언제든지 구독을 해지할 수 있습니다. 해지 시점 이후의 구독 기간에 대해서는 환불이 이루어지지 않습니다.
  • 제14조 (준거법): 본 약관은 대한민국 법률을 준거법으로 합니다.
  • 제15조 (광고 금지): 회사는 "원금 보장", "수익 보장" 등의 표현을 사용하지 않습니다.
  • 제16조 (API 사용): 회원은 거래소 API 이용약관도 함께 준수해야 합니다.
  • 제17조 (개인정보): API Key는 사용자 PC에만 저장되며, 회사는 이를 수집하지 않습니다.
  • 제18조 (업데이트): 업데이트는 수익률 보장을 의미하지 않습니다.
  • 제19조 (약관 변경): 약관 변경 시 7일 전 공지합니다.
